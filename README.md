# Flight Study App

A real-time 3D virtual flight study app.

This project combines:
- Python FastAPI backend
- Unity 3D client
- Blender-created assets
- Real-time flight data
- Study tools such as Pomodoro timer and cabin ambient sound

## Repository Structure

```txt
backend/   Python FastAPI backend
unity/     Unity client project
blender/   Blender source assets
docs/      Documentation
.github/   GitHub workflows
```

## Development Roadmap

1. Backend mock API
2. Unity placeholder aircraft movement
3. Real flight data integration
4. Study UX
5. Blender assets
6. Oracle Cloud deployment

## Backend Development

Initial backend commands will be documented after the FastAPI MVP is added.

## Unity Development

Create the Unity project under:

```txt
unity/FlightStudyClient
```

Do not commit Unity generated folders such as `Library/`, `Temp/`, `Obj/`, or build outputs.
